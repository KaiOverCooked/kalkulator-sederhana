# 🧮 Kalkulator Sederhana

Proyek kalkulator sederhana berbasis terminal yang dibuat dengan Python.  
Ini adalah project pertama saya untuk belajar dasar-dasar pemrograman Python!

## ✨ Fitur

- ➕ Penjumlahan
- ➖ Pengurangan
- ✖️ Perkalian
- ➗ Pembagian (dengan pengecekan pembagian nol)
- 🔄 Bisa digunakan berulang kali tanpa restart

## 🚀 Cara Menjalankan

Pastikan kamu sudah menginstal [Python 3](https://www.python.org/downloads/).

1. Clone repository ini:
   ```bash
   git clone https://github.com/USERNAME/kalkulator-sederhana.git
   ```

2. Masuk ke folder project:
   ```bash
   cd kalkulator-sederhana
   ```

3. Jalankan programnya:
   ```bash
   python calculator.py
   ```
